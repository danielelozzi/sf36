# Calcolatore Punteggio SF-36

Questo repository contiene un calcolatore online per il questionario sulla qualità della vita SF-36.

## Funzionalità

La pagina web (`index.html`) permette di:

*   Inserire le 36 risposte del questionario SF-36.
*   Inserire dati demografici come età e sesso.
*   Calcolare in tempo reale i seguenti punteggi:
    *   **Punteggi delle 8 scale** trasformati su una scala da 0 a 100.
    *   **Z-scores** standardizzati rispetto alla popolazione generale degli Stati Uniti.
    *   **Indici Sintetici di Salute Fisica (ISF/PCS)** e **Salute Mentale (ISM/MCS)**, presentati come T-scores basati sulle norme USA.
    *   **Punteggi T standardizzati per Età e Sesso** basati sulle norme della popolazione italiana.
*   Visualizzare i risultati tramite tabelle riassuntive e grafici interattivi.

Il calcolo dei punteggi 0-100 si basa sulla logica del calcolatore dell'Istituto di Ricerche Farmacologiche Mario Negri IRCCS.

## Utilizzo

1.  Aprire il file `index.html` in un qualsiasi browser web moderno.
2.  Compilare i campi del questionario e i dati demografici.
3.  I punteggi e i grafici verranno aggiornati automaticamente.
